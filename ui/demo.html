🐍 Snake Water Gun

A terminal-based Snake-Water-Gun game built in Python. 10 rounds per match, score tracking, and a replay prompt. No libraries — just run it.

Python 3.6+ Terminal Game stdlib only MIT License
$python main.py

Rules

Your Pick Beats Why
🐍 Snake 💧 Water Snake drinks it
💧 Water 🔫 Gun Water rusts it
🔫 Gun 🐍 Snake Gun shoots it

Features

🎮

10-Round Matches

Each game runs for exactly 10 rounds with live round-by-round output.

📊

Score Tracking

Win percentage displayed for both sides at the end of every match.

🔁

Replay Prompt

Play again without restarting the terminal — just type yes or no.

✅

Input Validation

Wrong input? The round replays cleanly instead of crashing.
